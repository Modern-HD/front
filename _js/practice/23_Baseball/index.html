게임 규칙

  1. 야구게임 기록을 서버에 저장하여 조회 및 삭제를 할 수 있게 만들어 볼 것
  2. 서버측의 데이터 키(객체의 키)는 수정이 불가능하므로 재해석이 필요함
  3. id: 자동생성, name: 게임 실행 유저 이름 or 컴퓨터의 정답
  4. description: 컴퓨터의 정답 or 유저가 시도한 내용 및 결과
  5. price: 소요된 시간, category_id: 총 시도 횟수 or 한 게임 동안 총 스트라이크 수
  6. created: 게임종료시간, modified: 자동 생성
  7. 게임 종료시 게임기록은 자동저장 or 버튼을 만들어 저장
  8. 게임기록 조회는 리스트로 삭제는 리스트의 각 라인 우측에 삭제버튼을 제작 => 클릭하면 해당 라인 삭제

제작 규칙

  • 1게임을 완료한 시간을 측정할 것
  • 게임을 진행하면서 숫자 추측 시도 횟수도 기록할 것
  • 페이지를 리프레쉬 하거나 내부 프로그램을 종료하기 전까지의 기록을 오름차순으로 정렬하여 출력할 것
